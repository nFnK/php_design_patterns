# PHP设计模式学习
一、OOP对象复习
二、创建型设计模式强调的是实例化过程。设计这些模式的目的是隐藏实例的创建过程，并封装对象使用的知识：
   1）抽象工厂Abstract Factory
   2）生成器Builder
   3）工厂方法Factory Method
   4）原型Prototype
   5）单例Singleton

三、结构型设计模式,重点是创建新结构而不破坏原有的结构
   1）适配器模式Adapter（类和对象）
   2）桥接模式Bridge
   3）组合模式Composite
   4）装饰器模式Decorator
   5）外观模式Pacade
   6）享元模式Flyweight
   7）代理模式Proxy
